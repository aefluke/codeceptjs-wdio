# sahibinden-ui

* Install requirements (*Nix):
    >sudo chmod 755 'nix_setup.sh' \
    >sudo ./nix_setup.sh

* Install requirements (MacOS):
    >sudo chmod 755 'macos_setup.sh' \
    >./macos_setup.sh

* Execute Local Docker Selenium Grid:
    >npm run codeceptjs:docker

* Execute Locally:
    >npm run codeceptjs
